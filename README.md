# C Programming for Beginners 🚀

A complete hands-on journey through the C programming language, from basic syntax to advanced topics such as data structures, networking, debugging, GUI development, scientific computing, and professional software development.

This repository contains practical examples, exercises, projects, and learning resources designed to help beginners become proficient C programmers.

## 🛠️ Requirements

* GCC Compiler
* Make
* GDB
* CMake (for advanced projects)
* GTK4 (for GUI projects)

### Fedora

```bash
sudo dnf install gcc make gdb cmake gtk4-devel
```

---

## 🚀 Compilation

Compile an individual file:

```bash
gcc filename.c -o program
./program
```

Compile projects using Make:

```bash
make
```
